Arcadia Lib

     


🇺🇸 English (US)

Arcadia Lib is the shared foundation required by other Arcadia mods developed by Aminoquiz (me).

🔧 What it provides

🗄️ MySQL Integration — HikariCP connection pool, configured via arcadia-database.toml.
👤 Player Data — daily streak tracking, particle preference storage, per-server tagging.
🪙 Arcadia Token — shared server currency item (arcadia_lib:arcadia_token) used by daily rewards and the auction house.
🛠️ Debug Mode — JVM flag -Darcadia.debug=true relaxes OP checks for local development.

⚙️ Setup

Configure your MySQL connection in config/arcadia-database.toml before starting the server.

[database]
host     = "localhost"
port     = 3306
name     = "arcadia_prestige"
user     = "root"
password = ""
max_pool_size = 10

🇫🇷 Français (FR)

Arcadia Lib est la bibliothèque partagée requise par tous les mods Arcadia. Elle fournit la connexion MySQL, la gestion des données joueurs et les utilitaires communs à la suite.

🔧 Ce qu'elle fournit

🗄️ Intégration MySQL — Pool HikariCP, configuré via arcadia-database.toml.
👤 Données Joueur — suivi des streaks, préférence de particule, tag multi-serveur.
🪙 Arcadia Token — monnaie serveur partagée (arcadia_lib:arcadia_token) utilisée pour les récompenses et l'hôtel des ventes.
🛠️ Mode Debug — Propriété JVM -Darcadia.debug=true pour le développement local.
