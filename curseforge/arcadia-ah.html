🇺🇸 English (US)

Arcadia Auction House is a cross-server player-driven marketplace. List items for sale, browse listings from other servers sharing the same database, and receive payments via a secure mailbox — even while offline.

✨ Key Features

🌐 Cross-Server — Listings from any server sharing the same MySQL database appear in every other server's AH.
📬 Mailbox System — Payments and returned expired listings are delivered to sellers on next login, even if they were offline or on a different server.
🔍 Search & Filter — Search by item name; filter by category (All / Pets / Misc) or view only your own listings.
🏆 Top Business Leaderboard — Ranks sellers by number of unique buyers.
💰 Numismatics Economy — Prices are paid and displayed using the Numismatics coin system.
⏱️ Configurable Expiry — Listings auto-expire after a configurable duration (default 48 hours) and are returned via mailbox.

🎮 Player Commands

/ahOpen the Auction House browser
/ah sell <price> [quantity]List the held item for sale at the given price
/ah topView the Top Business leaderboard

🛠️ Admin Commands

/ah enable / /ah disableEnable or disable the Auction House server-wide (OP level 2)

⚙️ Config (arcadia-ah.toml)

listing_duration_hoursHow long listings stay active before expiring (default: 48)
max_listings_per_playerMax simultaneous active listings per player (default: 30)

🇫🇷 Français (FR)

Arcadia Auction House est un marché joueur cross-serveur. Mettez des objets en vente, parcourez les annonces d'autres serveurs partageant la même base de données, et recevez vos paiements via une boîte aux lettres sécurisée — même hors ligne.

✨ Fonctionnalités Clés

🌐 Cross-Serveur — Les annonces de tous les serveurs connectés à la même base MySQL sont visibles partout.
📬 Boîte aux Lettres — Paiements et annonces expirées livrés à la reconnexion, même depuis un autre serveur.
🔍 Recherche & Filtres — Recherche par nom ; filtres par catégorie (Tout / Pets / Divers) ou vos propres annonces.
🏆 Classement Top Business — Classe les vendeurs par nombre d'acheteurs uniques.
💰 Économie Numismatics — Prix affichés et débités via le système de pièces Numismatics.
⏱️ Expiration Configurable — Les annonces expirent après une durée configurable (48h par défaut) et sont retournées par boîte aux lettres.

🎮 Commandes Joueur

/ahOuvrir l'hôtel des ventes
/ah sell <prix> [quantité]Mettre l'objet tenu en vente
/ah topVoir le classement Top Business

🛠️ Commandes Admin

/ah enable / /ah disableActiver ou désactiver l'hôtel des ventes (OP niveau 2)
